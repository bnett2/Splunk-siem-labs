# Splunk SIEM Labs
This repository documents hands-on Splunk SIEM labs focused on security monitoring,
log analysis, detection engineering, and incident triage.
## Environment
- Splunk Enterprise (Local Lab)
- Windows Event Logs
- Sysmon
- Sample attack simulations
## Skills Demonstrated
- SPL (Search Processing Language)
- Log ingestion and normalization
- Security detection use cases
- Alert creation and tuning
- Dashboard development
- Incident analysis and documentation
## Lab Index
- 00-setup/
- 01-log-ingestion/
- 02-search-spl/
- 03-dashboards-alerts/
- 04-use-cases-detections/
## Disclaimer
All data used is simulated or publicly available. No sensitive or production data is included.
